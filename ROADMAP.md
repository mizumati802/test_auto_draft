# test_auto_draft 開発ロードマップ

## 🛠 参照リソース
詳細：[INTEGRATION_MAP.md](./INTEGRATION_MAP.md) を参照。

## 🚀 プロジェクト目標
画像解析からテンプレート予測、タイトル・説明文の自動生成、そして出品ページへの自動流し込みまでを一気通貫で実行する「完全オートメーション出品」の構築。

## 📅 開発フェーズ

### Phase 1: 下書き判定とトリガーの確立 (完了)
- [x] 下書き編集ページ（/sell/draft/）の自動検知（SPA対応）。
- [x] ロード5秒後の「未記入判定」実行。
- [x] 3行以内の商品説明を「未記入（要自動入力）」と定義し、ポップアップで可視化。

### Phase 2: 自動流し込み（EasyRegisterロジックの統合）
- [ ] 判定結果が「3行以内（未記入）」の場合に自動トリガー。
- [ ] サムネイル（商品画像）の存在確認フラグの実装（画像解析の前提条件）。
- [ ] EasyRegisterのテンプレート・ハッシュタグ結合ロジックを呼び出し。
- [ ] 商品説明欄（textarea）への自動インジェクション（流し込み）。

### Phase 3: 画像解析・AIオートメーション（最終目標）
- [ ] 画像解析によるカテゴリ・属性の自動抽出（vintage_extend のロジックを採用）。
- [ ] マスターデータ照合によるガードレール（不該当データの排除）の実装。
- [ ] 最適なテンプレートの自動予測・プリセット。
- [ ] AIによる魅力的な商品名（タイトル）および商品説明文の自動生成。
- [ ] 人の手を介さない「一気通貫オートメーション」の完成。
